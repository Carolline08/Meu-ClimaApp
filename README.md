# Meu-ClimaApp

Aplicativo mobile desenvolvido em React Native com Expo que consome dados da API pública Open-Meteo para exibir informações climáticas em tempo real.

---

## Funcionalidades

- Clima atual
- Temperatura em tempo real
- Umidade do ar
- Velocidade do vento
- Previsão para 7 dias
- Busca por cidade
- Modo escuro
- Background dinâmico
- Interface moderna com cards

---

## Tecnologias utilizadas

- React Native
- Expo
- JavaScript
- Axios
- Open-Meteo API

---

## API utilizada

Open-Meteo:  
https://open-meteo.com/

---

## Instalação

Clone o repositório:

```bash
git clone https://github.com/seuusuario/MeuClima-App-react-native.git
```

Entre na pasta do projeto:

```bash
cd weather-app-react-native
```

Instale as dependências:

```bash
npm install
```

Inicie o projeto:

```bash
npx expo start
```
